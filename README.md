# 🤖 DonAI Chatbot

DonAI is a simple AI-powered chatbot built using **HTML, CSS, SCSS, and Python (Flask)**.  
It demonstrates frontend-backend integration for real-time conversational interfaces.

---

## 🚀 Features

- 💬 Interactive chatbot UI
- ⚡ Fast Flask backend
- 🎨 Styled with CSS & SCSS
- 🔗 Frontend + Backend integration
- 🧠 Easy to extend with AI models or APIs

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, SCSS
- **Backend:** Python (Flask)
- **Version Control:** Git & GitHub
